MicrotonVST for Windows and macOS

October 11, 2025 · One min read
Euclidean Instruments
Development team

We’re excited to announce the release of MicrotonVST for Windows and macOS!

MicrotonVST features all of the sonic possibilities of the original Microton along with the convenience of the VST3 format. Now Windows and Mac users can bring true microtonal music to any DAW or software that supports the VST3 standard.

The MicrotonVST also includes a standalone synth application for use outside of DAW software, so you can instantly turn your desktop into a self-contained microtonal synth.

MicrotonVST is DRM-free and includes all future updates and patches. Best of all, it’s currently available at the introductory price of $5.

An Introduction to the Microton tuning system

November 2, 2024 · 2 min read
Euclidean Instruments
Development team

Let's have some fun with the Microton's tuning system.

The Microton defaults to 12-tone equal temperament (or 12-TET for short) at A440. (Each element's frequency is exactly 100 cents from the previous element, using A4 = 400hz as a starting point.) When we click on each element on the monochord, we can see its properties in the Element Editor.
